# 04. 기술 스택 & 아키텍처 (하이브리드 전략)

> 노션: https://app.notion.com/p/38c7834b045e817ba54ac49ebfc6534e
> **전략: 단일 UI 셸(Flutter) + 부위별 네이티브 모듈 + 웹뷰 혼합.**
> ※ Flutter와 React Native를 동시에 UI로 운용하는 건 비현실적(빌드·유지보수 2배)이라 지양.
> 대신 "부위별로 가장 적합한 기술"을 섞는다.

## 🧩 부위별 프레임워크 매핑 (핵심)

| 영역 | 사용 기술 | 이유 |
| --- | --- | --- |
| **앱 셸 / 네비게이션** | **Flutter (Dart)** | 일관된 렌더링·고성능 애니메이션, 기준 프레임워크 |
| 커머스 탐색(홈·카테고리·상세) | Flutter | 이미지 그리드·커스텀 UI 완성도 |
| **AI 피팅 결과 UI** | Flutter | 결과 비교·갤러리 등 시각 비중 큰 화면 |
| **카메라·사진 업로드/전처리** | **네이티브 모듈(Swift/Kotlin)** + Flutter 플러그인 | 카메라·이미지 처리 성능과 OS 권한 안정성 |
| 결제·소셜로그인·푸시 SDK | 네이티브 SDK 연동(플러그인) | PG·카카오·애플·FCM 공식 SDK 안정성 |
| **콘텐츠성 화면(이벤트·기획전·약관·공지·FAQ)** | **웹뷰(WebView)** | 앱 심사 없이 즉시 업데이트, 운영 민첩성 |
| **AI 추론 파이프라인** | **서버(Python/FastAPI)** | 무거운 연산은 전부 서버에서 처리 |

> 원칙: 화면은 Flutter가 기본. 네이티브는 "성능·OS 의존" 부위만, 웹뷰는 "자주 바뀌는 콘텐츠"만.

## 📱 프론트(앱) — Flutter 셸

- **Flutter (Dart)** — 단일 코드베이스, 일관 렌더링(Impeller)
- 상태관리: **Riverpod** (대안: Bloc)
- 네비게이션: **go_router**
- 네트워크: **dio** + **retrofit**
- 애니메이션: 기본 위젯 + **Rive/Lottie**
- 웹뷰: **flutter_inappwebview** (네이티브 ↔ 웹 브리지)
- 카메라/이미지: **camera**, **image_picker** + 필요 시 자체 네이티브 채널

## ⚙️ 백엔드

- API 서버: **Python(FastAPI)** — AI 연계 용이
- DB: PostgreSQL + Redis(캐시·큐)
- 인증: 소셜 로그인(OAuth) + JWT
- 스토리지/CDN: S3 호환 + CloudFront (이미지 중심 서비스)

## 🧠 AI 파이프라인 (서버)

- 이미지 생성/편집: 외부 API 또는 자체 호스팅 모델 (예산·품질에 따라)
- 비동기 워커 큐: **Celery** (또는 BullMQ)
- 결과 캐싱 + 안전필터(NSFW·저작권) + 워터마크
- **프로바이더 추상화 레이어**로 모델 교체 가능하게 설계

## 📦 인프라 / 대시보드

- BaaS 빠른 시작: Supabase 또는 Firebase (Auth·DB·Storage·Push)
- 결제: 국내 PG(토스페이먼츠/포트원) + 인앱결제
- 푸시: FCM
- 웹뷰 콘텐츠 호스팅: 별도 웹(Next.js) 또는 CMS
- 분석: Amplitude/GA4, 크래시 리포팅(Sentry)

## 📐 아키텍처 개요

```
┌─────────────────── Flutter 앱 셸 ───────────────────┐
│  커머스 UI · AI 피팅 UI · 네비게이션                  │
│   ├─ 네이티브 모듈(카메라/결제/로그인/푸시)            │
│   └─ 웹뷰(이벤트·기획전·약관·FAQ)                     │
└───────────────┬─────────────────────────────────────┘
                │ REST/GraphQL
        ┌───────▼────────┐
        │  API (FastAPI) │ → [커머스 서비스]
        │                │ → [AI 서비스] → [워커 큐] → [이미지 모델]
        └───────┬────────┘
                └→ [PostgreSQL / Redis / S3·CDN]
```
