# 03. 핵심 기능 — AI 가상 피팅 (Virtual Try-On)

> 노션: https://app.notion.com/p/38c7834b045e811ca3bdd017cd570d79
> **이 기능이 제품의 핵심 차별점이다.**

## 🎯 기능 정의

사용자가 **내 반려동물 사진을 업로드**하면, 현재 상세페이지에서 보고 있는
**상품(옷/모자/하네스 등)을 입힌 실사 합성 이미지**를 생성한다.
강아지·고양이·토끼 등 종을 지원.

## 🔄 사용자 플로우

```
1. 상품 상세 → [우리 아이한테 입혀보기] 버튼
2. 펫 프로필 선택 (없으면 사진 업로드 + 종/체형 입력)
3. 생성 중 로딩 (애니메이션, 5~15초)
4. 결과: 옷 입은 실사 이미지 + [다시 생성] [저장] [공유] [장바구니]
5. 마음에 들면 바로 구매 / 갤러리에 보관
```

## 🧠 AI 파이프라인 (개념)

| 단계 | 처리 | 비고 |
| --- | --- | --- |
| 입력 정규화 | 펫 사진 누끼·자세·종 인식 | 품질 게이트(흐림/얼굴 미검출 거르기) |
| 상품 컨디셔닝 | 상품 이미지+메타(카테고리/색/소재) | 레퍼런스 이미지로 사용 |
| 생성 | 이미지 생성/편집 모델(try-on/inpainting) | 펫 정체성 유지가 핵심 |
| 후처리 | 업스케일, 안전필터, 워터마크 | 저작권/오남용 방지 |

> ⚠️ 난도 최상: '펫의 정체성(얼굴·무늬) 유지'와 '옷의 자연스러운 착용'을 동시에
> 만족시키는 것. 종/자세별 품질 편차가 큼.

## 🛡️ 품질·신뢰 장치

- 결과에 "AI 생성 이미지" 배지 명시 (실제 착용과 다를 수 있음 고지)
- 사이즈 추천은 AI 이미지가 아닌 **체중·둘레 실측 기반 표**로 별도 제공
- 생성 실패/저품질 시 자동 재시도 + 크레딧 미차감

## 💰 비용·운영 고려

- 1회 생성당 GPU/인퍼런스 비용 발생 → **무료 횟수 제한 + 크레딧/구독** 모델
- 결과 캐싱(같은 펫+같은 상품 재요청 시 재사용)
- 비동기 큐 처리로 트래픽 스파이크 대응

## 🔌 모델/API 옵션 (검증 대상 — P1 PoC)

- 상용 이미지 생성/편집 API (try-on 특화 또는 범용 inpainting)
- 자체 호스팅 오픈 모델 (비용↓, 운영 부담↑)
- **프로바이더 추상화 레이어**를 두어 모델 교체 가능하게 설계 (벤더 종속 최소화)

## 🧪 MVP 범위

- 강아지 1종부터 시작 → 정면 1자세, 상의 카테고리 한정
- 품질 기준 통과 후 종/자세/카테고리 점진 확대
