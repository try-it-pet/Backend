# 06. 고도화 / 개선 아이디어 (백로그)

> 노션: https://app.notion.com/p/38c7834b045e8156b8fce14781102a6d

## 🤖 AI 고도화

- 펫 멀티샷(여러 각도/자세) 업로드 → 더 정확한 피팅
- 영상/GIF 피팅 (걷는 모습에 옷 입히기)
- AI 스타일링: "우리 아이에게 어울리는 코디 추천"
- 체형·견종 기반 **AI 사이즈 추천** (반품률 감소)
- 얼굴 인식 기반 품종 자동 추정

## 🛒 커머스·비즈니스

- AI 피팅 결과 공유 → 바이럴 루프(신규 유입)
- 구독(피팅 무제한) / 크레딧 비즈니스
- 브랜드 입점·광고, 제휴 커머스
- 정기구독(사료·간식 자동배송)
- 맞춤 제작(프린팅) 연계

## 👥 커뮤니티·소셜

- 펫 프로필 피드(인스타형), 팔로우
- 피팅 코디 투표/랭킹·챌린지
- 모임·산책 지도(당근식 지역 기반)
- 중고·나눔 장터(안 맞는 옷 거래)

## 🛠️ 운영·품질

- AI 결과 품질 피드백 루프(좋아요/신고)로 모델 개선
- 상품 3D/AR 피팅(고급)
- 다국어·글로벌 확장
- 접근성(고대비·음성·큰 타겟영역)
