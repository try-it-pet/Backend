# 01. 앱 특징 & 기능 명세

> 노션: https://app.notion.com/p/38c7834b045e811ca3bdd017cd570d79

## ⭐ 앱의 핵심 특징

1. **AI 가상 피팅** — 내 반려동물에게 상품을 입힌 실사 이미지 생성 (핵심 차별점, `03_AI_TRYON.md`)
2. **빠른 렌더링 & 부드러운 UX** — 토스급 마이크로인터랙션, 스켈레톤·낙관적 UI
3. **반려동물 프로필 중심** — 종/품종/체중/사이즈 기반 개인화 큐레이션
4. **신뢰 커머스** — 리뷰·실측 사이즈표·AI 표기 등 투명성
5. **공유·바이럴** — 피팅 결과 SNS 공유로 신규 유입

## 🗂️ 기능 목록

### 커머스
- [ ] 상품 탐색(홈/카테고리/검색/필터)
- [ ] 상품 상세(이미지·옵션·리뷰·실측표)
- [ ] 장바구니 / 결제 / 주문 내역
- [ ] 찜(위시리스트)

### AI 가상 피팅 (핵심)
- [ ] 반려동물 사진 업로드·프로필화
- [ ] 상품 상세에서 "우리 아이한테 입혀보기" CTA
- [ ] 생성 결과 보기·재생성·저장·공유
- [ ] 결과 갤러리(여러 상품 비교)

### 반려동물 프로필
- [ ] 종(견/묘/토끼)·품종·체중·나이 등록
- [ ] 멀티 펫 관리
- [ ] AI 사이즈 추천 연동

### 계정·기타
- [ ] 소셜 로그인(카카오·애플·구글)
- [ ] 마이페이지·설정·알림
- [ ] 푸시 알림(찜 할인, 생성 완료 등)

## 🎚️ 우선순위 (MoSCoW)

| 등급 | 기능 |
| --- | --- |
| Must | AI 피팅(단순 1종), 상품 상세, 장바구니·결제, 펫 프로필 |
| Should | 사이즈 추천, 결과 공유, 찜, 리뷰 |
| Could | 결과 갤러리 비교, 커뮤니티 피드 |
| Won't(이번) | AR/3D 피팅, 영상 피팅, 다국어 |
