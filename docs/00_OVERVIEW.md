# PetFit (펫핏) — 반려동물 AI 쇼핑 플랫폼

> 반려동물(강아지·고양이·토끼)을 위한 모바일 쇼핑 플랫폼.
> 핵심은 **내 반려동물 사진을 올리면 상품(옷)을 입힌 실사 이미지를 AI로 생성**해주는 가상 피팅 기능.
> 당근·무신사·토스 수준의 렌더링 품질과 UX를 목표로 한다.

- **상태:** 🟡 기획 단계 (Planning)

## 🧩 문제 정의

- 사이즈·핏 불확실성 → 구매 망설임, 높은 반품률
- 모델 사진과 내 아이의 느낌 차이 → 실망
- 상품 탐색이 공급자(카테고리) 중심 → 우리 아이 기준 큐레이션 부재

## 💡 해결책(비전)

내 반려동물 사진 1장으로, 현재 보고 있는 옷·용품을 입힌 실사 이미지를 즉석 생성 →
구매 결정을 감성·시각적으로 지원.

## 👥 타깃 사용자
