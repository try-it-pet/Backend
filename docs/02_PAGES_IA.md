# 02. 페이지 / 화면 설계 (IA)

> 노션: https://app.notion.com/p/38c7834b045e818b92e3e459a0ac41fb

## 🧭 하단 탭 (5탭)

| 탭 | 이름 | 역할 |
| --- | --- | --- |
| 1 | 홈 | 큐레이션·추천·기획전 |
| 2 | 카테고리/검색 | 종·아이템별 탐색 |
| 3 | **AI 피팅** (중앙 CTA) | 핵심 기능 진입점 |
| 4 | 좋아요/갤러리 | 찜·저장한 피팅 |
| 5 | 마이페이지 | 펫 프로필·주문·설정 |

## 📱 화면 목록

### 온보딩
- 스플래시
- 로그인(카카오·애플·구글)
- 펫 프로필 등록: 종 → 품종 → 체중·나이 → 사진 업로드

### 커머스
- 홈 (피드·추천·배너)
- 카테고리 리스트 / 검색 결과 / 필터
- 상품 상세 (이미지·옵션·리뷰·실측표·**피팅 CTA**)
- 장바구니 / 결제 / 주문 완료

### AI 피팅
- 피팅 진입(펫 선택) → 생성 로딩 → 결과 → 결과 갤러리(비교)

### 마이페이지
- 프로필·펫 관리·주문내역·찜·포인트·설정

## 🧩 디자인 원칙 (당근·무신사·토스 벤치)

- 한 화면 = 한 개의 핵심 행동 (토스식 단순함)
- 엄지손 존(thumb zone)에 주요 CTA 배치
- 스켈레톤 · 낙관적 빈상태 · 즉각 피드백(햅틱)
- 이미지 중심 그리드(무신사식), 일관된 여백·타이포

## 🔗 핵심 전환 경로

```
홈/상세 → [우리 아이한테 입혀보기] → 결과 → 장바구니 → 결제
                                  └→ 공유(바이럴) → 신규 유입
```
