# 07. 디자인 시스템 (Design System)

> **목적: 모든 화면을 일관된 룩앤필로 디자인하기 위한 단일 기준(Single Source of Truth).**
> 새 화면을 클로드/디자인 도구로 만들 때 **이 문서를 통째로 컨텍스트로 제공**하면 일관성이 유지된다.
> 맨 아래 [복붙용 디자인 프롬프트](#-복붙용-디자인-프롬프트-클로드용) 블록을 화면 요청 앞에 붙여 사용.

---

## 🎨 브랜드 무드

- 키워드: **다정함 · 신뢰 · 프리미엄 · 부드러움**
- 톤: 따뜻하지만 깔끔한 모던 커머스. 과한 장식 X, 이미지가 주인공.
- 레퍼런스 감성: 당근의 따뜻함 + 무신사의 이미지 큐레이션 + 토스의 단순함.

## ✍️ 타이포그래피

- **폰트: Pretendard** (한글 최적화, 숫자 가독성 우수). 영문/숫자 동일 패밀리.
- 줄간격(line-height) 기본 1.4, 제목은 1.25.

| 토큰 | 크기/굵기 | 용도 |
| --- | --- | --- |
| `display` | 28 / Bold | 큰 타이틀·온보딩 |
| `h1` | 22 / Bold | 화면 제목 |
| `h2` | 18 / SemiBold | 섹션 제목 |
| `body` | 15 / Regular | 본문 |
| `body-strong` | 15 / SemiBold | 강조 본문 |
| `caption` | 13 / Regular | 보조 설명 |
| `label` | 12 / Medium | 칩·태그·뱃지 |
| `price` | 18 / Bold | 가격 (강조) |

---

## 📏 스페이싱 & 레이아웃

- **4px 베이스 그리드.** 모든 간격은 4의 배수.
- 스케일: `4 · 8 · 12 · 16 · 20 · 24 · 32 · 40`
- 화면 좌우 기본 패딩: **16px**
- 컴포넌트 간 세로 간격: 기본 **12~16px**, 섹션 간 **24~32px**
- 콘텐츠 최대폭: 모바일 100%, 태블릿 600px 중앙 정렬

## 🔘 라운딩 & 그림자

| 토큰 | 값 | 용도 |
| --- | --- | --- |
| `radius-sm` | 8 | 칩·작은 버튼 |
| `radius-md` | 12 | 카드·인풋 |
| `radius-lg` | 16 | 큰 카드·바텀시트 |
| `radius-full` | 999 | 원형 버튼·아바타 |
| `shadow-card` | y2 blur8 rgba(0,0,0,0.06) | 카드 |
| `shadow-float` | y4 blur16 rgba(0,0,0,0.10) | 떠 있는 CTA·모달 |

## 🧱 핵심 컴포넌트 규칙

### 버튼
- **Primary**: 배경 `primary`, 텍스트 흰색, 높이 **52px**, `radius-md`, `body-strong`. 풀폭이 기본.
- **Secondary**: 배경 `gray-100`, 텍스트 `ink`.
- **Outline**: 보더 `gray-300`, 배경 투명.
- 눌림 시 `primary-pressed` + 살짝 축소(scale 0.98) + 햅틱.
- 비활성: `gray-300` 배경, `gray-500` 텍스트.

### 카드 (상품)
- 배경 `bg`, `radius-md`, `shadow-card`.
- 구성: 정사각 이미지 → 브랜드(caption) → 상품명(body) → 가격(price) → 찜 아이콘.
- 이미지 비율 1:1, `object-fit: cover`.

### 하단 탭바
- 5탭, 높이 56px + 세이프에어리어. 중앙 **AI 피팅**은 도드라진 원형 FAB 스타일(`primary`).
- 활성 `primary`, 비활성 `gray-500`.

### 인풋
- 높이 48px, `radius-md`, 보더 `gray-300`, 포커스 시 `primary`.
- placeholder `gray-500`.

### 칩 / 필터
- 높이 32px, `radius-full`, 선택 시 `primary-light` 배경 + `primary` 텍스트.

### AI 생성 배지
- `ai-badge`→`ai-gradient-end` 그라데이션, 흰 텍스트 "AI", `label` 크기, `radius-full`.
- AI로 만든 모든 이미지에 **반드시** 표기.

### 빈 상태 / 로딩
- 로딩: 스켈레톤(`gray-100` 셰이드 애니메이션).
- 빈 상태: 일러스트 + 한 줄 안내 + 행동 버튼.

---

## 🐾 일러스트 / 아이콘

- 아이콘: **라인 스타일**, 2px 스트로크, 둥근 끝, 24px 기본.
- 일러스트: 따뜻한 파스텔, 둥근 형태. 강아지·고양이·토끼 모티프.
- 사진: 자연광·실사 톤 유지. 과한 필터 지양.

---

## ♿ 접근성

- 본문 대비 최소 4.5:1, 큰 텍스트 3:1.
- 터치 타겟 최소 **44x44px**.
- 색만으로 정보 전달 금지(아이콘/텍스트 병행).
- 다크모드: 추후 토큰 다크 변형 정의 (P2 이후).

## 📋 복붙용 디자인 프롬프트 (클로드용)

> 새 화면을 디자인 요청할 때, 아래 블록을 복사해 맨 앞에 붙이고 화면 설명을 이어 쓰세요.

```
[PetFit 디자인 시스템 적용]
- 무드: 다정함·신뢰·프리미엄·부드러움. 이미지가 주인공, 장식 최소화.
- 컬러: primary #FF6B3D, primary-light #FFE9E1, accent #1FB6A8,
  텍스트 ink #1A1A1A / gray-500 #8A8A8A, 배경 #FFFFFF / bg-warm #FFF9F5,
  AI 배지 그라데이션 #7C5CFF→#1FB6A8.
- 폰트: Pretendard. display28B/h1 22B/h2 18SB/body 15R/caption 13R/price 18B.
- 스페이싱: 4px 그리드, 화면 좌우 패딩 16, 섹션 간 24~32.
- 라운딩: 카드/버튼/인풋 12, 큰카드 16, 칩/원형 full.
- 버튼: Primary 풀폭 52px, radius 12, 눌림 #E5532A.
- 그림자: 카드 y2 blur8 6%, 떠있는 요소 y4 blur16 10%.
- 하단탭 5개, 중앙 'AI 피팅'은 primary 원형 FAB.
- AI 생성 이미지엔 'AI' 배지 필수.
- 접근성: 터치타겟 44px+, 대비 4.5:1.

위 시스템을 지켜서 [화면 이름] 화면을 디자인해줘. 구성요소: [...]
```
